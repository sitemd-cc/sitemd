## About this demo

This site is built entirely with sitemd — a markdown site builder for humans and coding agents working together.

Every page is a `.md` file. Every setting is a `.md` file. No HTML, no CSS, no JavaScript — just markdown.

Open any file in your editor to see how it works, or ask your coding agent to make changes for you.
