# Welcome to your site

This is a fresh sitemd project. Edit this page or create new ones in the `pages/` directory.

## Getting started

1. Edit `settings/meta.md` to set your site name and description
2. Edit `settings/header.md` to configure navigation
3. Create new pages as `.md` files in `pages/`
4. Run `sitemd launch` to preview your site

## Learn more

Visit [sitemd.cc/docs](https://sitemd.cc/docs) for full documentation.
